MÚSICA DE FUNDO
================

Coloque um arquivo chamado "background.mp3" NESTA pasta para habilitar a
música de fundo do DigitAI.

O player do app tenta carregar `/music/background.mp3`. Se o arquivo não
existir, o app continua funcionando normalmente (sem música).

Sugestões de onde baixar música infantil livre de direitos:

1. Pixabay Music (gratuito, sem atribuição obrigatória)
   https://pixabay.com/music/search/kids/

2. Bensound (gratuito para uso não comercial com atribuição)
   https://www.bensound.com/royalty-free-music/childrens

3. Incompetech (Kevin MacLeod, CC-BY, atribuição necessária)
   https://incompetech.com/music/royalty-free/

4. FreePD (domínio público)
   https://freepd.com/

Recomendado: trilha instrumental calminha, 2-3 minutos (o player dá loop),
tamanho até ~2 MB para não pesar o site. Ex.:
  - "Happy Kids" - Pixabay
  - "Little Idea" - Bensound
  - "Wallpaper" - Kevin MacLeod

Depois de baixar, renomeie para `background.mp3` e coloque aqui.
Faça commit + push — o GitHub Pages vai servir o arquivo automaticamente.
